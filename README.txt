Any information about a movie in IMDB ?
This is the program to run.

This is a python script which uses the already existing OMDB API 
which is a collection of IMDB movies and information about it 
store in a JSON format.The user requests for information about 
a particular movie in this program.The program access the 
information in a JSON format and depending upon the title the user
requests the user gets further specific information about the movie.
The user must have already installed JSON in python to run this program.
